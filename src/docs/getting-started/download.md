---
title: Download
---

You can either download the package using [Bower][bower] with running:

```
$ bower install mark.js --save-dev
```

Or using [npm][npm]:

```
$ npm install julmot/mark.js --save-dev
```

Or download the package [manually]({{{defaults.download}}}).

[bower]: http://bower.io/
[npm]: http://npmjs.com/
